How the WebApp works

Process Data

We process data from the Question Classification Dataset, getting it all to a standard format

Feature Extraction

We convert every word in the corpus into a feature using TF-IDF Vectorization. We use these features for Prediction

Build Model

A powerful Machine Learning model is built and the parameters are tuned using GridSearch for the best performance.

Predict Results

The models are pickled and saved for prediction APIs, and are served using a Flask Server straight to this WebApp.
